# LightStrike

El objetivo de este proyecto es entrenar la coordinación óculo-manual (ojo-mano) del usuario.
Para ello, hemos creado una espada con cortadora láser e impresora 3D para un videojuego desarrollado
con Arduino y Processing.La espada contendrá una tira led y LDRs. El processing generará figuras que
serán mostradas mediante un proyector y el usuario deberá de alinear la espada con la figura antes de
un periodo de tiempo determinado. Si logra alinear la espada a tiempo ganará un punto y si no perderá vida.
Los LDRs se encargarán de detectar la luz de la figura proyectada y si todos los leds están en verde, es
que la espada está perfectamente alineada. Para añadir dificultad, la espada tendrá dentro pequeñas bolsas
llenas de arroz para añadir peso y las figuras se moverán.
